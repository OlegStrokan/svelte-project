

# svelte app

Test application written in svelte

## Get started

Install the dependencies...

```bash
cd svelte-app
npm install
```

```bash
npm run dev
```

Navigate to [localhost:5000](http://localhost:5000)
## Production mode

To create an optimised version of the app:

```bash
npm run build
```

## Using TypeScript

```bash
node scripts/setupTypeScript.js
```
